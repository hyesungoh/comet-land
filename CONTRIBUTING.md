# Welcome to Comet-land contributing guide

Thank you for investing your time in `Coment-land` !

> Read our [Code of Conduct](https://github.com/hyesungoh/comet-land/blob/main/CODE_OF_CONDUCT.md) to keep our community approachable and respectable.

## Comet-land wants ...

- New feature.
- Reporting or Fixing bug.
- Discussing current code.
- Append new tests.
- Update or translate documents.
- and everything what you think of Comet-land need it.

## How to contribute

1. Fork the repo.
2. Do what you want for Comet-land.
3. Make sure checking tests and lint.
4. Issue that pull request !
